# AI 知识科普：面向软件工程师的深度学习指南

本项目是一系列专门为软件开发工程师编写的 AI 科普文章。我们不堆砌深奥的数学公式，而是尝试将抽象的 AI 概念映射到你熟悉的编程模型（如 MapReduce、调用栈、单元测试、OOM、if-else 等）中，带你直击深度学习的底层逻辑。

## 🚀 为什么读这些文章？

- **实事求是**：内容严谨，包含大量对常见误区的勘误（如数学公式、历史事实、拼写错误）。
- **极客视角**：用工程师的直觉解构模型（卷积层 = Map，全连接层 = Reduce）。
- **硬核深度**：不仅讲“是什么”，更讲“为什么”（例如 ReLU 为什么能解决梯度消失，而非仅仅是非线性激活）。

## 📚 目录

1. **[面部识别技术解析](1面部识别.md)**
   - 核心概念：CNN 卷积神经网络、ReLU 激活函数（梯度消失解决方案）、Dropout（过拟合对策）。
   - 类比：将 Sobel 算子理解为滑窗式的特征过滤器。

2. **[AI 眼中的文字](2AI眼中的文字.md)**
   - 核心概念：BPE 分词、Word2Vec 静态向量 vs Transformer 动态向量、余弦相似度。
   - 类比：Embedding 本质上是一个巨大的权重查找表（Look-up Table）。

3. **[多层叠加计算器](3多层叠加计算器.md)**
   - 核心概念：卷积层 vs 全连接层、神经网络四大门派分类。
   - 类比：神经网络就像是一个堆叠了无数层、能自动调优的超级计算器。

4. **[电脑是如何学习的](4电脑是如何学习的.md)**
   - 核心概念：损失函数、梯度下降、反向传播（链式法则）。
   - 类比：梯度下降就像是在高维空间中寻找最优解的混沌工程实验。

5. **[Attention 机制解析](5Attention.md)**
   - 核心概念：Transformer 的核心组件——注意力机制。
